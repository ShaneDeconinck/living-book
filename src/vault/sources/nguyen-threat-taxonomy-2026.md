**Citation:** Nguyen et al., "A Taxonomy of Agentic AI Security Threats and Multi-Agent Framework Security Analysis," arXiv:2603.09002, March 9, 2026.

**Method:** Catalog every documented agentic AI security threat, classify across a nine-category taxonomy, score sixteen existing frameworks on coverage per category.

**Key findings:**
- 193 distinct threats identified
- Nine threat categories: identity/authentication, authorization/access control, data integrity, model behavior/reliability, tool/plugin security, communication/coordination, privacy, regulatory compliance, non-determinism
- OWASP Agentic Security Initiative leads all 16 frameworks at 65.3% overall coverage
- [[non-determinism-gap]]: mean coverage 1.231/5 (most under-addressed)
- Data leakage: mean coverage 1.340/5 (second most under-addressed)
- First empirical cross-framework comparison for multi-agent system security
